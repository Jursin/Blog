### 🛠️ 实用应用
- **[MifareClassicTool](https://github.com/ikarus23/MifareClassicTool)** - 用于读取、写入、分析等的 Android NFC 应用
- **[Breezy Weather](https://github.com/breezy-weather/breezy-weather)** - 功能丰富的免费开源 Material 3 Expressive 天气应用程序
- **[RikkaHub](https://github.com/rikkahub/rikkahub)** - 支持多个 LLM 提供商的 Android 应用
- **[ReadYou](https://github.com/ReadYouApp/ReadYou)** - 小巧的安卓 RSS 阅读器

### 📂 文件与应用相关
- **[flut-renamer](https://github.com/sun-jiao/flut-renamer)** - 用 Flutter 编写的批量文件重命名应用
- **[Aves](https://github.com/deckerst/aves)** - 使用 Flutter 编写的图库和元数据浏览应用
- **[LocalSend](https://github.com/localsend/localsend)** - 局域网文件传输应用
- **[AltSendme](https://github.com/tonyantony300/alt-sendme)** - 一个跨平台 p2p 文件传输工具
- **[HybridFileXfer](https://github.com/weixiansen574/HybridFileXfer)** - 多轨快传，同时使用 USB、5G 与 2.4G WiFi 等通道传输文件到电脑
- **[LibChecker](https://github.com/LibChecker/LibChecker)** - 用于查看设备中应用程序所用库的应用
- **[Obtainium](https://github.com/ImranR98/Obtainium)** - 支持直接从多种渠道获取安卓应用更新的应用
- **[TaskManager](https://github.com/RohitKushvaha01/TaskManager)** - 适用于安卓系统的任务管理器

### 📥 下载管理器
- **[Aria2App](https://github.com/devgianlu/Aria2App)** - 基于 aria2 的高级下载管理器
- **[Gopeed](https://github.com/GopeedLab/gopeed)** - 快速、现代的下载管理器，支持 HTTP、BitTorrent、Magnet 和 ed2k 下载
- **[Seal](https://github.com/JunkFood02/Seal)** - 视频/音频下载器

### 📝 编辑器与开发工具
- **[Acode](https://github.com/Acode-Foundation/Acode)** - 功能强大的安卓文本/代码编辑器
- **[Xed-Editor](https://github.com/Xed-Editor/Xed-Editor)** - 功能全面且可扩展的 Android 文本编辑器
- **[Termux](https://github.com/termux/termux-app)** - 一款十分强大的终端模拟器应用
- **[ProxyPin](https://github.com/wanghongenpin/proxypin)** - HTTP(S) 流量捕获应用
- **[Flutter Server Box](https://github.com/lollipopkit/flutter_server_box)** - 使用 Flutter 提供图表显示 Linux/Unix/Windows 服务器状态并进行管理
- **[Shizuku](https://github.com/RikkaApps/Shizuku)**
- **[Dhizuku](https://github.com/iamr0s/Dhizuku)**
- **[aShellYou](https://github.com/DP-Hridayan/aShellYou)** - 具有 Material Design 3 用户界面的全功能 Android Shell 应用

### 🔐 安全与密码管理
- **[Keyguard](https://github.com/AChep/keyguard-app)** - Bitwarden® 平台和 KeePass (KDBX) 的替代客户端
- **[Stratumauth](https://github.com/stratumauth/app)** - 适用于 Android 和 Wear OS 的双因素身份验证 (2FA) 客户端

### 🎨 第三方客户端
- **[Pilipala](https://github.com/guozhigq/pilipala)** - 使用 Flutter 开发的 BiliBili 第三方客户端
- **[PiliPalaX](https://github.com/orz12/PiliPalaX)** - 基于 Pilipala 做了更激进的修改的 Flutter BiliBili 客户端
- **[PiliPlus](https://github.com/bggRGjQaUbCoE/PiliPlus)** - 基于 PilipalaX 做了更激进的修改的 BiliBili 客户端
- **[BiliPai](https://github.com/jay3-yy/BiliPai)** - 原生、纯净、可扩展的第三方 Bilibili Android 客户端
- **[TiebaLite](https://github.com/zzc10086/TiebaLite)** - 非官方的贴吧客户端
- **[zhihu-plus-plus](https://github.com/zly2006/zhihu-plus-plus)** - 去广告、占用低、AI大模型的新时代知乎安卓端
- **[AuroraStore](https://github.com/whyorean/AuroraStore)** - 允许下载 Google Play 商店中的应用的非官方商店
- **[Droid-ify](https://github.com/Droid-ify/client)** - F-Droid 第三方客户端
- **[Neo Store](https://github.com/NeoApplications/Neo-Store)** - F-Droid 第三方客户端
- **[pixez-flutter](https://github.com/Notsfsssf/pixez-flutter)** - 支持免代理直连及查看动图的第三方 Pixiv Flutter 客户端
- **[Olib](https://github.com/shiyi-0x7f/olib-mobile)** - Z-Library 第三方客户端

- **[legado Plus](https://github.com/Luoyacheng/legado)** - 继承自 legado，在其基础上新增更多功能
- **[Animeko](https://github.com/open-ani/animeko)** - 集找番、追番、看番的一站式弹幕追番平台
- **[Kazumi](https://github.com/Predidit/Kazumi)** - 基于自定义规则的番剧采集 APP
- **[Venera](https://github.com/venera-app/venera)** - 支持阅读本地和网络漫画的应用
