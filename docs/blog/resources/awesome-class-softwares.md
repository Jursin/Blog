> 因缺乏动力且需要更专注于大学生活，**Awesome-Class-Softwares** 基本停更，网页版也会下线并重定向到该博客页面。不过同类项目 [Awesome Iwb](https://github.com/awesome-iwb/awesome-iwb) 收集得更全，且[网页版](https://aiwb.smart-teach.cn/)已上线，你仍可以通过该项目了解更多软件。

<!-- more -->

- 课程表类
  - [SmartTeachCN/CsesWebEditor](https://github.com/SmartTeachCN/CsesWebEditor) - 一个基于 Web 的课程表编辑器，全称 Course Schedule Exchange Schema，支持 CSES 格式的课程表创建、管理和导出
  - [==ClassIsland/ClassIsland=={.tip}](https://github.com/ClassIsland/ClassIsland) - 一款适用于班级多媒体屏幕的课表信息显示工具
  - [Class-Widgets/Class-Widgets](https://github.com/Class-Widgets/Class-Widgets) - 一款使用 Python 开发的 Qt 风格的全新桌面课表软件
  - [Yamikani-Flipped/LingYun-Class-Widgets](https://github.com/Yamikani-Flipped/LingYun-Class-Widgets) - 一款功能丰富的桌面工具，支持在桌面显示各种组件，支持 CSES 通用课程表格式
  - [EnderWolf006/ElectronClassSchedule](https://github.com/EnderWolf006/ElectronClassSchedule) - 一款电子课程表软件
  - [GPUawa/iClass](https://github.com/GPUawa/iClass) - 一个使用 Electron 和 Vue 3 构建的教室大屏实用工具箱

- 多功能类
  - [==STBBRD/ZongziTEK-Blackboard-Sticker=={.tip}](https://github.com/STBBRD/ZongziTEK-Blackboard-Sticker) - 用于白板一体机的桌面小部件，集成了课程表、天气提醒、快速黑板、启动台等功能
  - [Ris-Soft/Ris_ClassTool](https://github.com/Ris-Soft/Ris_ClassTool) - 瑞思课堂工具框架
  - [OrbiBoard/OrbiBoard](https://github.com/OrbiBoard/OrbiBoard) - 基于 Electron 的以 JavaScript 为核心的插件化大屏课堂辅助工具
  - [==laoshuikaixue/VoiceHub=={.tip}](https://github.com/laoshuikaixue/VoiceHub) - 一个使用 Nuxt 4 全栈框架开发的现代化校园广播站点歌管理系统
  - [PANDAJSR/sidebar-for-class](https://github.com/PANDAJSR/sidebar-for-class) - 一个基于 Electron 开发的课堂侧边栏应用程序

- 看板类
  - [==ExamAware/ExamAware2=={.tip}](https://github.com/ExamAware/ExamAware2) - 一款易用、跨平台的大屏考试信息展示工具
  - [ExamAware/ExamShowboard-Legacy](https://github.com/ExamAware/ExamShowboard-Legacy) - 旧版考试看板
  - [ExamAware/ExamSchedule](https://github.com/ExamAware/ExamSchedule) - 静态网页版本的考试看板，可自行部署
  - [StickyHomeworks2/StickyHomeworks2](https://github.com/StickyHomeworks2/StickyHomeworks2) - 一款支持富文本的桌面作业贴工具
  - [Xwei1645/zooni](https://github.com/Xwei1645/zooni) - 一个教室大屏作业看板，基于 Flutter 开发，支持多平台
  - [==ZeroCatDev/Classworks=={.tip}](https://github.com/ZeroCatDev/Classworks) - 基于 Vue 3 + Vuetify 的现代化作业管理系统前端项目
  - [EnderWolf006/HomeworkBoard](https://github.com/EnderWolf006/HomeworkBoard) - 一款支持云同步的作业展示板软件
  - [==QQHKX/Immersive-clock=={.tip}](https://github.com/QQHKX/Immersive-clock) - 一款基于 React + TypeScript + Vite 构建的轻量化桌面/网页时钟应用
  - [Return-Log/Education-Clock](https://github.com/Return-Log/Education-Clock) - 一个实用的班级信息看板，支持课程表、倒计日、天气预报、公告板等功能
  - [Candlest/ClassBoard](https://github.com/Candlest/ClassBoard) - 一款高自由度的动态壁纸软件，支持高考倒计时、公告栏、课程表等功能
  - [cjhdevact/LockTime](https://github.com/cjhdevact/LockTime) - 一个具有深/浅色主题、云母/亚克力背景、Fluent UI 界面的时钟锁屏（屏保）
  - [xinghai-smartedu/classwindow](https://github.com/xinghai-smartedu/classwindow) - 一个基于 Electron 构建的桌面悬浮窗软件，用于显示班级信息
  - [SECTL/assignsticker](https://github.com/SECTL/assignsticker) - 一个使用 Pywebview 开发的简洁、易用的作业看板软件

- 批注类
  - [==Alan-CRL/Inkeys=={.tip}](https://github.com/Alan-CRL/Inkeys) - Windows 屏幕批注工具，适用于触摸设备和 PC 端，支持多指书写、荧光笔、PPT 批注等功能
  - [WXRIW/Ink-Canvas](https://github.com/WXRIW/Ink-Canvas) - 一款轻量级画板软件，基于 WPF/C#，针对希沃一体机特别优化，启动速度提升 5-10 倍
  - [clover-yan/Ink-Canvas-Plus](https://github.com/clover-yan/Ink-Canvas-Plus) - 复刻自 Ink Canvas 的 Windows 画板应用，优化了各方面使用体验
  - [==WuChanging/Ink-Canvas=={.tip}](https://github.com/WuChanging/Ink-Canvas) - 基于 Ink Canvas 二次开发，优化了界面 UI，是一款好用的屏幕批注/板书/PPT 批注工具
  - [BaiYang2238/Ink-Canvas-Better](https://github.com/BaiYang2238/Ink-Canvas-Better) - 在 Ink-Canvas-Artistry 基础上二次修改，移除了非画板核心功能
  - [InkCanvas/InkCanvasForClass](https://github.com/InkCanvas/InkCanvasForClass) - 一款开源易用的电子教鞭软件，专为课堂教学场景优化设计
  - [==InkCanvasForClass/community=={.tip}](https://github.com/InkCanvasForClass/community) - InkCanvasForClass 社区版
  - [SECTL/Kazuha](https://github.com/SECTL/Kazuha) - 适用于智慧黑板/交互智能平板的演示助手工具
  - [wwiinnddyy/LanStartWrite](https://github.com/wwiinnddyy/LanStartWrite) - 基于 Electron 的新一代书写应用
  - [lh11117/LemonxNote](https://github.com/lh11117/LemonxNote) - 一款用 C++ 和 Qt 开发的轻量白板软件

- 辅助类
  - [HugoAura/Seewo-HugoAura](https://github.com/HugoAura/Seewo-HugoAura) - 下一代希沃管家注入式修改方案
  - [SRInternet-Studio/Seewo-Custom_Start](https://github.com/SRInternet-Studio/Seewo-Custom_Start) - 希沃白板定制启动器
  - [SRInternet-Studio/Class_Hot_Search](https://github.com/SRInternet-Studio/Class_Hot_Search) - 适用于班级大屏的排行榜和公共投票小工具
  - [hxabcd/EasiAuto](https://github.com/hxabcd/EasiAuto) - 一款自动登录希沃白板的小工具
  - [RinLit-233-shiroko/PowerPoint-Touch-Assist](https://github.com/RinLit-233-shiroko/PowerPoint-Touch-Assist) - 在 Windows 10/11 中更高效地以"单击"动作翻页的 PPT 触屏辅助工具
  - [jiugulixiaoniu/ClassScreenLock](https://github.com/jiugulixiaoniu/ClassScreenLock) - 一个 Windows 系统课堂屏幕锁定软件
  - [SECTL/SecScore](https://github.com/SECTL/SecScore) - 一个简单易用、优雅美观的班级个人积分管理软件
  - [SECTL/ShowWrite](https://github.com/SECTL/ShowWrite) - 一个简洁优雅的视频展台工具，专为教学演示设计
  - [WSXYT/PrismCore](https://github.com/WSXYT/PrismCore) - 一个基于 WinUI 3 构建的 Windows 系统优化工具，提供内存管理、CPU 智能调度、延迟诊断、磁盘清理、网络修复和启动项管理等功能
  - [WSXYT/C30Auto-login](https://github.com/WSXYT/C30Auto-login) - 一个基于图像识别与自动化控制的 C30 教学软件自动登录工具

- 抽选类
  - [==SECTL/SecRandom=={.tip}](https://github.com/SECTL/SecRandom) - 一款能以多种方式进行完全公平的随机抽取的软件
  - [NamePickerOrg/NamePicker](https://github.com/NamePickerOrg/NamePicker) - 一款简洁的点名软件
  - [LuoYunXi0407/Rand](https://github.com/LuoYunXi0407/Rand) - 一款美观、易用、强大的抽号软件
  - [typed-sigterm/ex-caller](https://github.com/typed-sigterm/ex-caller) - 一个简约风格的随机点名工具
