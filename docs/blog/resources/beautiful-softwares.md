> 收录一些实用的软件，涵盖文件管理、日常工具、系统维护、开发工具等。

<!-- more -->

- 文件相关
  - [Everything](https://www.voidtools.com/zh-cn/) - 基于名称快速定位文件和文件夹
  - [File Locator Pro](https://www.mythicsoft.com/filelocatorpro/) - 一款高效的文件搜索软件
  - [ReNamer](https://www.den4b.com/products/renamer) - 一款高效的文件重命名软件
  - [Once Power](https://github.com/ilgnefz/once_power) - 基于 Flutter 开发的文件批量重命名、批量移动删除工具
  - [FastCopy](https://fastcopy.jp/) - Windows上最快的复制/备份软件
  - [Bandizip](https://www.bandisoft.com/bandizip/) - 功能强大的压缩软件
  - [7zip](https://www.7-zip.org/) - 一款开源压缩软件
  - [LocalSend](https://localsend.org/zh-CN) - 开源多平台文件局域网传输工具
  - [LANDrop](https://landrop.app/) - 跨平台局域网文件传输工具
  - [FileZilla](https://filezilla-project.org/) - 一款开源、免费的 FTP 解决方案

- 日常工具
  - [zTasker](https://www.everauto.net/) - 一款定时任务软件
  - [AutoDarkMode](https://github.com/AutoDarkMode/Windows-Auto-Night-Mode) - 在预定时间切换 Windows 的暗色和亮色主题
  - [Textify](https://ramensoftware.com/textify) - 从不提供此类功能的对话框中复制一些文本
  - [QuickLook](https://github.com/QL-Win/QuickLook) - 将 MacOS "快速查看"功能引入 Windows
  - [VLC media player](https://www.videolan.org/vlc/) - 一款自由、开源的跨平台多媒体播放器及框架，可播放大多数多媒体文件，以及 DVD、音频 CD、VCD 及各类流媒体协议。
  - [PotPlayer](https://potplayer.tv/?lang=zh_CN) - 一款免费的多媒体播放器
  - [Motrix Next](https://motrix-next.pages.dev/) - 一款功能齐全的下载管理器
  - [Internet Download Manager](https://www.internetdownloadmanager.com/) - 一个下载加速器
  - [Neat Download Manager](https://neatdownload.com/neat-download-manager/) - 一个下载加速器
  - [Ghost-Downloader-3](https://gd.xychr.com/) - 基于人工智能的下一代跨平台多线程下载器
  - [SmartSystemMenu](https://github.com/AlexanderPro/SmartSystemMenu) - 扩展系统中所有窗口的系统菜单
  - [PixPin](https://pixpin.cn/) - 功能丰富的截图软件
  - [EcoPaste](https://ecopaste.cn/) - 免费、开源的跨平台剪贴板管理工具
  - [Pot](https://pot-app.com/) - 一款跨平台划词翻译和OCR软件
  - [ApowerREC](https://www.apowersoft.cn/record-all-screen) - 多平台录屏工具
  - [Bandicam](https://www.bandicam.cn/) - 一款高清的电脑录屏软件
  - [OBS Studio](https://obsproject.com/zh-cn) - 免费且开源的用于视频录制以及直播串流的软件
  - [Geek Uninstaller](https://geekuninstaller.com/) - 好用的卸载软件
  - [BCUninstaller](https://www.bcuninstaller.com/) - 批量垃圾卸载程序

- 信息查询
  - [图吧工具箱](https://tb.saop.cc/) - 硬件检测工具合集
  - [图吧工具箱 WinUI3 版](https://tubawinui3.cn/) - 图吧工具箱的重构版，基于 WinUI 3 / .NET 10 全新打造
  - [CPU-Z](https://www.cpuid.com/softwares/cpu-z.html) - 显示处理器相关信息
  - [GPU-Z](https://www.techpowerup.com/gpuz/) - 显示显卡相关信息
  - [CrystalDiskInfo](https://crystalmark.info/en/software/crystaldiskinfo/) - 查看磁盘详细信息
  - [CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark/) - 一个简单的磁盘基准测试软件
